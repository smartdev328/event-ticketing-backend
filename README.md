## Project setup

```bash
$ npm install
```

Run this command to migrate database tables
```bash
$ npm run db:migrate
```

Run seed command to seed data
```bash
$ npm run seed
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
